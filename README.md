# MES Futures Scalping Strategy

Personal algorithmic trading strategy developed in C# using NinjaTrader for Micro E-mini S&P 500 futures.

## Overview

This project was created to learn more about automated trading logic, market analysis, and risk management using NinjaTrader.

The strategy uses technical indicators to identify market conditions and support automated trading decisions.

## Features

- EMA trend filtering
- ATR volatility analysis
- Bollinger Bands integration
- Automated risk management logic
- Profit target and stop-loss conditions
- Trade execution rules

## Technologies

- C#
- NinjaTrader
- NinjaScript

## Project Status

This project is currently under active development and testing.

Future improvements may include backtesting results, performance tracking, and additional strategy refinements.

## Purpose

Built as a personal learning project to strengthen my understanding of algorithmic trading systems, automation logic, and software development using C#.
